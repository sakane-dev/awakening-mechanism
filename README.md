# AWAKENING-MECHANISM | COGNITIVE_INTERFACE

https://pages.sakane.dev/awakening-mechanism

## 覚醒機構Webサイトデザイン案 No.1

### 概要

認知科学と言語哲学の知見を背景に、サイバーパンクな世界観と高い学術的信頼性を融合させた「覚醒機構」のインターフェースを構築しました。

デザインシステム: Oklch(0.1 0 0)の漆黒背景に、Oklch(0.8 0.15 160)のサイバーネオングリーンの発光（グロウ効果）を定義しました。
シネマティック・グラスモーフィズムと、極めて細い高透過ボーダーを組み合わせることで、UIが空間に浮遊しているような深度を与えています。
タイポグラフィ: 日本語は Noto Sans JP の軽量ウェイト、英数字やシステムコードは JetBrains Mono を採用。
太字を排除し、情報の階層をフォントサイズと不透明度で制御することで、洗練されたテック感と学術的な誠実さを演出しています。
インタラクション: ターミナルエミュレータを模したグラスモーフィズム・コンテナを配置し、静的な表示ではなく、意味のないハッシュ文字列から「観測者がシステムを定義する。
認知の境界を再構築中...」へと徐々にデコードされるタイピングエフェクトを想起させる設計にしています。
これにより、ユーザーがシステムの中枢にアクセスしているような処理流暢性を演出し、高度な技術力を直感させるUIを実現しました。
